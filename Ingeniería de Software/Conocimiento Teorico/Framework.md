### Starter app

Una *starter app* (aplicación de inicio) es una **plantilla o estructura base preconfigurada** utilizada por desarrolladores para acelerar la creación de nuevas aplicaciones, incluyendo componentes esenciales, código base y herramientas de configuración iniciales

### CodeIgniter 4

Es un framework PHP ligero y rápido que incluye por defecto una estructura [[MVC]] completa, enrutamiento flexible (con rutas manuales por defecto), soporte para bases de datos, validación de formularios, gestión de sesiones, manejo de errores y un sistema de caché

---
### Diferencia entre una aplicación y un Framework

| Framework             | Starter App                          |
| --------------------- | ------------------------------------ |
| Es la estructura base | Es una aplicación inicial ya montada |
| No es ejecutable solo | Sí es ejecutable                     |
| Define reglas         | Usa esas reglas                      |
| Es genérico           | Es personalizable                    |
#### El framework controla:
- HTTP Request 
	
- Router
	
- Controller
	
- Model
	
- Database
	
- View
	
- HTTP Response

#### La Starter App es donde tú escribes:
- Controladores
    
- Modelos
    
- Vistas
    
- Reglas de negocio
